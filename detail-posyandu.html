Aplikasi Website Posyandu Kartika

Website informatif untuk mencegah stunting

Tanggal: Oktober 2023
Peran: Frontend Developer, UI Designer
Kategori: Web Development, Kesehatan

Ikhtisar Proyek

Aplikasi Website Posyandu Kartika adalah platform digital yang dirancang untuk membantu orang tua dan petugas kesehatan dalam memantau pertumbuhan dan perkembangan anak, khususnya dalam upaya pencegahan stunting. Proyek ini berfokus pada penyediaan informasi yang mudah diakses dan alat pemantauan yang intuitif.

Tantangan

Tantangan utama dalam proyek ini adalah bagaimana menyajikan data kesehatan yang kompleks (seperti kurva pertumbuhan) agar mudah dipahami oleh orang tua awam, serta memastikan data tersebut aman dan akurat. Integrasi dengan basis data yang efisien juga menjadi fokus penting.

Solusi & Teknologi

Proyek ini dikembangkan menggunakan kombinasi teknologi modern untuk frontend dan backend:

  • Frontend: Dibangun dengan React.js untuk antarmuka pengguna yang dinamis dan responsif, didukung oleh Bootstrap 5 untuk styling yang konsisten dan cepat.
  • Backend: Menggunakan Node.js dengan framework Express.js untuk API yang efisien dan skalabel.
  • Database: MySQL digunakan untuk penyimpanan data pertumbuhan anak, riwayat kunjungan, dan informasi kesehatan lainnya.
  • Alur Kerja: Mengimplementasikan metodologi agile untuk pengembangan yang iteratif dan kolaboratif.

Teknologi Digunakan

React.js Node.js Express.js MySQL Bootstrap 5 Figma (UI/UX) Git

Fitur Utama

  • Pemantauan Pertumbuhan Anak (Berat, Tinggi)
  • Dashboard Petugas Kesehatan
  • Pencatatan Imunisasi
  • Edukasi Gizi untuk Orang Tua
  • Antarmuka Pengguna Responsif
  • Sistem Notifikasi Jadwal Posyandu
