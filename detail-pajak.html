Aplikasi Web Perhitungan Pajak Pegawai

Sistem Manajemen Pajak Pegawai Dinas Kominfo Pringsewu

Tanggal: 2024
Peran: Fullstack Developer
Kategori: Web Development, Keuangan, Pemerintahan

Ikhtisar Proyek

Aplikasi Web Perhitungan Pajak Pegawai ini dikembangkan untuk mendukung Divisi Keuangan Dinas Kominfo Pringsewu dalam mengelola dan menginput data pajak pegawai secara digital. Sistem ini memudahkan proses pencatatan, perhitungan, dan pelaporan pajak pegawai dengan pengelolaan data yang terintegrasi dan aman.

Tantangan

Tantangan utama adalah membangun sistem autentikasi yang aman dan mudah digunakan untuk dua tipe pengguna (admin & pegawai), serta memastikan data pajak yang diinputkan dapat tersimpan dan terhubung antar tabel (relasi NIP) di database phpMyAdmin secara efisien.

Solusi & Teknologi

Proyek ini dikembangkan menggunakan:

  • Backend & Framework: PHP dengan CodeIgniter 4 (CI4) untuk struktur MVC yang rapih dan scalable.
  • Frontend: Bootstrap 5 untuk tampilan modern dan responsif.
  • Database: MySQL (phpMyAdmin) dengan relasi utama pada field NIP antara tabel admin dan user pegawai.
  • Keamanan: Sistem login untuk pegawai (harus mendaftar NIP terlebih dahulu) dan admin (akun sudah dibuatkan khusus).

Teknologi Digunakan

PHP CodeIgniter 4 Bootstrap 5 MySQL (phpMyAdmin) HTML5 CSS Github

Fitur Utama

  • Login & Registrasi Pegawai (registrasi NIP, password)
  • Login Admin (admin sudah dibuatkan oleh sistem)
  • Input Data Pajak Pegawai
  • Penyimpanan data pajak ke database ter-relasi
  • Dashboard Admin: Kelola data pegawai & pajak
  • Dashboard User: Riwayat Arsip & status pajak pribadi
  • Validasi dan keamanan data
  • Dua tampilan: Admin & User (pegawai)
