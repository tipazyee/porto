Beans & Bites

Aplikasi web modern untuk pemesanan kopi dan dessert secara digital

Tanggal: Januari 2025
Peran: Fullstack, UI Designer
Kategori: Web Development, Food & Beverage, E-Commerce

Ikhtisar Proyek

Beans & Bites adalah aplikasi berbasis web yang dirancang untuk memudahkan pelanggan dalam melihat menu kopi dan dessert, melakukan pemesanan secara online, serta menyelesaikan pembayaran dengan cepat dan praktis. Website ini memberikan pengalaman digital yang nyaman bagi pelanggan tanpa harus login, sekaligus memberikan kemudahan bagi pemilik coffee shop dalam mengelola menu dan stok secara efisien melalui dashboard admin.

Tantangan

Menyediakan sistem pemesanan online yang ringan, intuitif, dan aman tanpa memaksa pengguna untuk login, serta memastikan pengelolaan produk oleh admin tetap efisien dan responsif.

Solusi & Teknologi

Website ini dibangun menggunakan stack modern:

  • Frontend menggunakan React.js untuk pengalaman pengguna yang dinamis dan cepat.
  • Backend dikembangkan dengan Pyramid (Python) untuk struktur server-side yang aman dan fleksibel.
  • Desain antarmuka dirancang menggunakan Figma agar estetika dan usability optimal.

Teknologi Digunakan

React.js Python Pyramid Figma Bootstrap Git

Fitur Utama

  • Tampilan Beranda dengan menu unggulan dan info toko
  • Halaman Menu kategori Coffee, Non-Coffee, dan Dessert
  • Keranjang Belanja untuk menambah, menghapus, dan mengubah jumlah produk
  • Checkout dan pembayaran melalui QRIS, E-Wallet, dan Transfer Bank
  • Konfirmasi pembayaran dan cetak struk digital
  • Login Admin untuk akses dashboard
  • Dashboard Admin untuk manajemen produk dan stok
  • Fitur CRUD produk oleh admin
